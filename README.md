## Project Overview

The Banking Transaction Analytics Dashboard is a Power BI solution designed to analyze customer transactions, account balances, customer demographics, and banking activity trends. The dashboard helps financial institutions monitor transaction patterns, understand customer behavior, evaluate account performance, and identify inactive accounts.

The report combines transaction and customer data into an interactive dashboard that enables stakeholders to track key banking metrics, compare account types, and gain insights into customer engagement across different demographic groups.

## Dataset Information

The dashboard uses a consolidated banking dataset named CombinedBankingDataset containing customer, account, and transaction information.

## Main Fields Used
(a) Customer Information : CustomerID, Name, Gender, Customer Age Group
(b) Account Information :  AccountType, Balance
(c) Transaction Information : TransactionDate, TransactionType, Amount
(d) Time Attributes : Year, Month
(e) Banking Metrics : Total Balance, Transaction Amount, Transaction Count

## Key Measures
- Count of Transactions
- Monthly Transaction Amount
- Total Balance
- Inactive Accounts
- Customer Count by Gender
- Account Count by Type
- Monthly Transaction Balance

### Dashboard Features

## Page 1: Transaction & Account Performance Analysis

- Pie chart showing the proportion of different transaction types. Helps identify the most frequently used banking services.
- Area chart displaying transaction amounts over time. Useful for detecting seasonal patterns and transaction growth.
- Bar chart comparing transaction amounts across customers. Highlights high-value customers and transaction contributors.
- Clustered column chart showing total balances by account type. Enables performance comparison between different account categories.
- Line chart tracking inactive accounts over time. Helps identify customer disengagement trends.

## Page 2: Customer Demographics & Account Insights

- Donut chart visualizing customer count by gender. Provides demographic breakdown of the customer base.
- Column chart showing customer distribution across age groups. Helps understand customer segmentation.
- Treemap representing account counts by account type. Identifies the most popular account categories.
- Bar chart comparing balances across customers. Highlights customers with significant account holdings.
- Area chart showing changes in account balances over time. Assists in monitoring overall banking portfolio growth.

### Key Insights
## Transaction Behavior
Transaction type analysis reveals the most commonly used banking activities.
Monthly trends help identify peak transaction periods and customer activity patterns.

## Customer Segmentation
Gender and age-group analysis provide a clear understanding of customer demographics.
Supports targeted marketing and customer engagement strategies.

## Account Performance
Account-type comparisons highlight which products generate the highest balances.
Treemap visualization identifies the most adopted account categories.

## Customer Value Identification
Customer-level balance and transaction analysis helps identify high-value customers.
Supports personalized banking services and retention programs.
## Operational Monitoring
Inactive account tracking enables early identification of dormant customers.
Helps improve customer retention and account activation initiatives.
